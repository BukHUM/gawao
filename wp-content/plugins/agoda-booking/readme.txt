Search and book hotels through Agoda Affiliate API. Integrate Agoda hotel search functionality into your WordPress site.

== Description ==

Agoda Booking is a WordPress plugin that integrates with Agoda Affiliate API to provide hotel search and booking functionality on your website.

**Features:**

* Search hotels by city or specific hotel list
* Display hotel information including prices, ratings, and images
* Filter results by price, star rating, and review score
* Redirect to Agoda for booking
* Configurable default language and currency
* Caching for improved performance
* Responsive design

**Requirements:**

* WordPress 5.8 or higher
* PHP 7.4 or higher
* Agoda Site ID and API Key

== Installation ==

1. Upload the `agoda-booking` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Agoda Booking to configure your Site ID and API Key
4. Use the shortcode `[agoda_search]` to display the search form on any page or post

== Frequently Asked Questions ==

= Do I need an Agoda account? =

Yes, you need to register as an Agoda affiliate and obtain your Site ID and API Key.

= Where can I get my Site ID and API Key? =

Contact your Agoda account manager or visit the Agoda Developer Portal.

= Can I customize the search form? =

Yes, you can customize the form appearance through CSS or by modifying the template files.
